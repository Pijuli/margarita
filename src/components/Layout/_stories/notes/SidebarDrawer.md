# Sidebar Drawer

Margarita has a componenet that allow us to display a sidebar

## Properties

#### Type
 Default -> `attached`
 Prop type: `String`
 Accepted sidebar types: `attached` and `fixed`

 #### Position
 Default -> `left`
 Prop type: `String`
 Accepted sidebar positions: `left` and `right`

 #### Shadow
 Default -> `true`
 Prop type: `Boolean`

 #### Show
 Default -> `false`
 Prop type: `Boolean`
